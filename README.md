## Eslint settings
1. Import order
2. No magic numbers
3. Naming convention

### Install
```
# npm
npm install
# yarn 
yarn
```

### Start
```
# npm
npm run dev
# yarn 
yarn dev
```

### Build
```
# npm
npm run build
# yarn 
yarn build
```

### Format
```
# npm
npm run format
# yarn 
yarn format
```

### Lint
```
# npm
npm run lint
# yarn 
yarn lint
```
